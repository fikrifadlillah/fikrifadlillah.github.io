Test Answers

Apa yang kamu ketahui tentang Version Control?

Suatu mekasnisme yang mencatat perubahan satu atau kumpulan file sepanjang waktu yang dimana versi spesifik / versi sebelumnya dari file dapat diakses kembali

Jelaskan apa itu git?

Software yang mengelola perubahan dalam kode sumber selama pengembangkan software untuk mengkoordinasikan pekerjaan dianara programmer dan daoat dimanfaatkan juga untuk melacak perubahan pada kumpulan file

Apa yang kamu ketahui tentang Framework khususnya Web Framework, sebutkan apa saja web framework yang kamu ketahui?

Rancangan software yang digunakan untuk mendukung pengembangan aplikasi web dengan aturan pemrograman yang telah ditentukan. Web Framework saat ini adalah Laravel, Codeigniterm dan Django

Jelaskan istilah - istilah berikut: REST API, SQL, HTTP, UI Framework !

  • REST API : Kumpulan fungsi-fungsi yang menggunakan protokol HTTP untuk mengkases atau memanipulasi data
  • SQL : Bahasa Terstruktur untuk menerima dan memanipulasi data pada database relasional
  • HTTP : Protokol pada lapisan jaringan yang digunakan untuk mengakses aplikasi web
  • UI fRAMEWORK : Kerangka tampilan/ / antarmuka web yang telah dilengkapi dengan Javascript yang telah dicustom untuk memanipulasi halaman HTML

Apa yang dimaksud dengan RDBMS dan noSQL, jelaskan perbedaannya!

  • RDBMS : Sistem basis data yang terdiri dari kumpulan tabel yang mempunyai relais dari satu tabel ke tabel lain
  • no SQL : Sistem Basis Data yang tidak mengikuti aturan database relasional dan umumnya bersifat document-oriented

Perbedaan antara RDMBS dan NoSQL adalah Sifat pendefinisian skema, yang dimana RDBMS bersifat permanen /static sedangkan noSQl bersifat dinamis, RDBMS lebih cocok untuk menangani query yang kompleks, dan noSQl lebih cocok untuk menangani penyimpanan yang bersifat hirarkis
